== Description ==

This plugin places a small button on the defined corner of your site which allows to scroll up the page. Button appears only if page is being scrolled. 

== Installation ==

1. Unzip and upload `wp-scrollup.zip` to the `/wp-content/plugins/wp-scrollup` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check up the settings to set up your button view.
